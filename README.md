## Showcase
![Alt Text](https://github.com/DarrahK/Dockerised-Neural-Network-ODE-Solver/blob/master/assets/example.gif)

## Usage
### Build
```
docker compose build
```

> **_NOTE:_**  May take a while to build

### Run
```
docker compose up
```

### View
Go to `http://localhost:8501/`
